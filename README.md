# SCB Kingston Website

A custom-designed website developed for the Society of Conservation Biology (SCB) Club at Kingston University.

[![Live Website](https://img.shields.io/badge/demo-live-brightgreen)](https://scbkingston.ca/)

![demo video](./assets/SCBKingstonDemo.gif)

---

## Overview

The Society of Conservation Biology Club at Kingston University wanted a modern website that better reflected their mission while remaining easy for club members to maintain after development.

Working directly with the club throughout the project, I designed multiple website concepts in Figma before developing a fully custom WordPress theme tailored to their vision. Choosing WordPress as the content management system allows future committee members to easily update content without requiring programming knowledge, ensuring the site remains maintainable for years to come.

---

## Features

- **Custom WordPress Theme** — Built from scratch to match the club's branding and vision rather than relying on a pre-made template.
- **Collaborative Design Process** — Designed several concepts in Figma and refined them through regular meetings and feedback sessions with club members.
- **Dynamic User Experience** — Enhanced the WordPress site with custom HTML and CSS to create modern layouts, animations, and interactive elements.
- **Consistent Multi-Page Design** — Designed and developed 7+ pages with a cohesive visual identity while allowing each page to communicate its own purpose and story.
- **Easy Content Management** — Implemented using WordPress so future club executives can update content without technical experience.

---

## Tech Stack

**CMS:** WordPress

**Design:** Figma

**Frontend:** HTML, CSS

---

## What I Learned / Challenges

One of the biggest challenges was creating a modern, engaging website while working within the club's limited budget. Without relying on premium plugins or advanced JavaScript frameworks, I used custom HTML and CSS to add interactive elements and create a more dynamic user experience. This required a great deal of research, experimentation, and creative problem-solving.

My favourite part of the project was the complete creative process, from the initial brainstorming sessions and Figma mockups to seeing the final website come together. Collaborating closely with the client throughout development taught me how valuable iterative feedback is in turning ideas into a finished product that truly matches a client's vision.
